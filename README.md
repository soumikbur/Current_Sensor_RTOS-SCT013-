\# Current Sensor RTOS (SCT013)



STM32 RTOS-based current monitoring system using SCT013 current sensor.



\## Features

\- Real-time current monitoring

\- FreeRTOS task scheduling

\- ADC-based signal acquisition



\## Hardware

\- STM32F410RB

\- SCT013 Current Sensor



\## Tools

\- STM32CubeIDE

\- FreeRTOS
